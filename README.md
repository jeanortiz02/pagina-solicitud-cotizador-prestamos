# Crediflash

Proyecto final del Grupo #5 para Taller de Programacion II.

## Estructura

- `index.html`: contenido principal de la pagina.
- `styles/styles.css`: estilos responsive.
- `js/app.js`: logica del cotizador, menu movil y formulario.
- `images/`: imagenes usadas por la web.

## Funciones incluidas

- Cotizador de prestamos convertido de React a JavaScript puro.
- Formulario de solicitud de prestamos.
- Navegacion por secciones.
- Diseno adaptable para computadoras y celulares.

Para abrirlo, usa el archivo `index.html` en un navegador.
